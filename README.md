# BofA Loan Calculator

Internal loan calculator tool for Bank of America. Supports mortgage, auto, and personal loan calculations with full amortization schedules.

## Tech Stack

- Angular 14.2.x
- Angular Material 14.x (legacy/pre-MDC components)
- TypeScript 4.7
- SCSS
- Karma/Jasmine for unit tests

## Prerequisites

- Node.js 16.x (required for Angular 14)
- Angular CLI 14.2.x

## Development Server

```bash
npm install
ng serve
```

## Build

```bash
ng build
```

## Running Unit Tests

```bash
ng test --watch=false
```
